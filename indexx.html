How to Play This Memory Test

1. Memorize

In each level, a number will appear on the screen. You have exactly 5 seconds to commit it to memory.

2. Recall

Once the timer hits zero, the number disappears. Type the exact sequence into the input box provided.

3. Progress

There are 20 levels in total. The numbers get longer (up to 20 digits!) as you advance through the test.

Scoring & Performance

Our Memory Test uses a weighted scoring system to reward players who reach the difficult higher levels.

Points are awarded based on the level number multiplied by 5:

  • Level 1: 5 Points
  • Level 10: 50 Points
  • Level 20: 100 Points
  • Maximum Total Score: 1050 Points

Rankings: 840+ is considered Excellent. Scores over 1000 are reserved for Memory Masters.
